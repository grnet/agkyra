Agkyra User Guide
=================

Agkyra is a syncing client for Pithos_ Storage. It is a multi-platform,
stand-alone, end-user application. It synchronizes the contents of local user
directories with remote Pithos containers through a
`Graphical User Interface (GUI)` as well as a `Command Line Interface (CLI)`.

Agkyra is implemented and supported by the `Synnefo Development Team`_ of the
`Greek Research and Technology Network`_ and is an open source project,
licensed under GPL v3.
